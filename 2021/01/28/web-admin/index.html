Spring Boot Admin 应用监控

1. 简介

Spring Boot Admin 可以帮助我们非常方便得查看应用的：

  • 健康信息
  • 内存指标
  • 日志信息
  • JVM 系统变量和环境变量
  • 线程和堆栈信息
  • 等等

还能进行状态通知用于告警，集成非常简单，简单又好用。
